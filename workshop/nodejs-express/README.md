# Workshop with NodeJS and Express


## Initial project
```
$npm init
$npm i -S express
$npm i -S dotenv
$npm i -S sequelize
$npm i -S pg
$npm i -S redis
$npm i -S joi

$npm i -D nodemon
$npm i -D jest
$npm i -D supertest
```

## Start with dev mode
```
$npm run dev
```

## Run test
```
$npm run test
$npm run test:coverage
```
